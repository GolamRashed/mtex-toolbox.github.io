mtexdata

load of data provided with mtex and often used in documentation

Syntax 

mtexdata        % displays a list of available loading routines
mtexdata name   % loads specified data set

Input

param Options that control the mtexdata behavior
Parameter Description
'aachen'
'3d'

serial section 3d EBSD data from Leo Kestens
'mylonite'

collected by Daniel Rutte (Brad R. Hacker)
'epidote'

data provided by David Mainprice
'dubna'

collected by Florian Wobbe at Dubna
'ptx'
'geesthacht'
