eig3

eigenvalue and vectors of a symmetric 3x3 matrix

Syntax 

lambda = eig3(M)
lambda = eig3(a11,a12,a13,a22,a23,a33)
[v,lambda] = eig3(a11,a12,a13,a22,a23,a33)

Input

M

array of symmetric 3x3 matrix
a11, a12,a13,a22,a23,a33

vector of matrix elements

Output

lambda

eigen values
v

eigen vectors
