subsref

(method of grainBoundary)

access subsets of a GrainSet

Syntax 

gB(1:10)          % the 10 first boundaries
grains('Fe')          % only Fe grains
grains( ~grains('fe') ) % all grains but Fe
                        logical array with size of the complete
                        GrainSet
