project2FundamentalRegion

(method of orientation)

projects orientation to a fundamental region

Syntax 

[ori,omega] = project2FundamentalRegion(ori,rot_ref)

Input

ori

orientation
ori_ref

reference rotation

Output

ori

orientation
omega

rotational angle to reference rotation
