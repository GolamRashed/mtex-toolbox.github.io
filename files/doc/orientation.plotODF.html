plotODF

(method of orientation)

Plot EBSD data at ODF sections

Syntax

plotODF(o,'param1',val1,'param2',val2)

Input

ebsd

EBSD
param,val Parameters and values that control plotODF
Parameter Description
'SECTIONS'

number of plots
'points'

number of orientations to be plotted
'all'

plot all orientations
'phase'

phase to be plotted
param Options that control the plotODF behavior
Parameter Description
'SIGMA (default)'
'OMEGA'

'GAMMA'

'PHI2'
'AXISANGLE'
