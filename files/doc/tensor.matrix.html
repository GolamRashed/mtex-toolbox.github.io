matrix

(method of tensor)

return tensor as a matrix

Syntax 

m = matrix(T)
m = matrix(T,'voigt')

Input

T

tensor
param,val Parameters and values that control matrix
Parameter Description
'voigt'

give a 4 rank tensor in voigt notation, i.e. as a 6 x 6 matrix

Output

m

matrix
