dot

(method of Miller)

inner product between two Miller indece

Syntax 

a = dot(m1,m2)
a = dot(m1,m2,'antipodal')

Input

m1,m2

Miller
param,val Parameters and values that control dot
Parameter Description
'antipodal'

consider m1,m2 with antipodal symmetry
'all'

Output

d

double [length(m1) length(cs)]
