smooth

(method of EBSD)

smooth spatial EBSD

Syntax

[ebsd,filter] = smooth(ebsd,'param1',val1,'param2',val2)

Input

ebsd

EBSD

Example 

mtexdata forsterite
ebsd = ebsd('indexed');
% segment grains
[grains,ebsd.grainId] = calcGrains(ebsd)
 
grains = grain2d

% find largest grains
largeGrains = grains(grains.grainSize>800)
