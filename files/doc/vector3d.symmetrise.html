symmetrise

(method of vector3d)

symmetrcially equivalent directions and its multiple

Syntax

[v,l,sym] = symmetrise(v,S,'param1',val1,'param2',val2)

Input

v

vector3d
S

symmetry
param Options that control the symmetrise behavior
Parameter Description
'antipodal'

include antipodal symmetry

'skipAntipodal'

do not include antipodal symmetry

Output

Sv

symmetrically equivalent vectors
l

number of symmetrically equivalent vectors
